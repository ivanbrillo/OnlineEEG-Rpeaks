Expected .mat File Structure

Each subject requires two MATLAB files. The fields listed below must be present with the exact names shown.

P###_EEG.mat

FieldType / ShapeDescription
EEG.data double (channels × timepoints) Raw EEG signal matrix. Channels in rows, time in columns.
EEG.srate scalar [Hz] Sampling rate

P###_ECG.mat

FieldType / ShapeDescription
ECG_i double vector [mV] ECG signal. One value per sample, same length as EEG timepoints.
R_peak integer vector Sample indices of R-peaks. 1-indexed (MATLAB convention) - converted to 0-indexed internally.
